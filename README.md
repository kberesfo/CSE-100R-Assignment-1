# Advanced Data Structures Template

This is a template repo for CSE 100R Advanced Data Structures 

## Merge Template

first clone:

```
git clone <instructor-lab-repo-url>
```

then merge template:

```
git remote add template https://github.com/kberesfo/CSE-100R-template.git
git fetch template
git merge template/main --allow-unrelated-histories
```

```
git remote remove template
```
